## Project 1 — Personal Finance Tracker

A simple web app to track your income, expenses, and savings with a monthly dashboard and basic AI insights.

### Features
- Add, update, and delete transactions
- Categorize as Income, Expense, or Savings
- Monthly summary dashboard with charts
- Search and filter transactions
- AI spending insights — top category, month-over-month trend, and next month's expense prediction

### How to Run
pip install streamlit pandas plotly
cd personal-finance-tracker
streamlit run app.py

## Project 2 — Smart Task Manager

A task management app with a Kanban board, notifications, and AI-powered suggestions.

### Features
- Create and manage tasks with title, description, priority, and due date
- Kanban board view — To Do, In Progress, Done
- Overdue and due-today notifications
- AI priority suggestion based on urgency keywords and due date
- AI task breakdown — generates subtask checklist for any task

### How to Run
pip install streamlit
cd smart-task-manager
streamlit run app.py
